Mirage 1.2

World's first instantaneous content adaptive brightness control app

Key Features:

  • Instantaneous content adaptive brightness control
  • Dynamic color temperature based on content
  • Seamless and non-intrusive transitions
  • Bright-white content aware dimming
  • User-friendly profile system
  • Separable settings for each connected monitor
  • On/Off setting per app (from tray icon)
  • Light-weight and very low resources consumption

System Requirements:

  • Windows 10 x64 or later
  • Admin permission for installation
  • Recommend for switching off that "content-aware brightness control" in Windows 11, since you will no longer need it.
  • Hardware: The software has been tested and run smooth on:
    • Laptop with i5 gen 8th + UHD 620
    • Laptop with i5 gen 12th + Iris-Xe

    If you have system of similar grade or above, expect the software to run smooth.

    Not yet have chance to test on an AMD though. Neither for 4k monitor nor 120fps display.
